
## SorHS

 This is sorting algorithms implementation using Haskell. The available algorithms(function name) are as follows. 

- mergeSort
- quickSort
- bubbleSort
- selectionSort
- insertionSort

## Usage 

```haskell

import Sort

ls = [4,3,4,6,8]
sorted = mergeSort ls
-- [3,4,4,6,8]

```
